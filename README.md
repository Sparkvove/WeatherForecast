To Run UI get into Weather.UI folder and:
<br>
1.install node (npm)
<br>
https://nodejs.org/en/download/
<br>
2. Then you need to install angular
<br>
`npm install -g @angular/cli`
<br>
3. Run UI
<br>
`ng serve - o`
<br>

To Run API, open WeatherAPI.sln , and run it.
